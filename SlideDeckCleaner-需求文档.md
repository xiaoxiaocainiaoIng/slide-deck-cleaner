# Slide Deck Cleaner - 需求文档

## 1. 产品定位

**产品名称**：Slide Deck Cleaner  
**核心功能**：批量清除 PPTX 文件中的水印、Logo 和冗余元素  
**目标用户**：经常使用 NotebookLM、AI PPT 工具生成演示文稿的用户

---

## 2. 功能需求

### 2.1 核心功能（F）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 水印检测 | 自动检测幻灯片中的水印元素（文字/图片） | P0 |
| 水印清除 | 移除检测到的水印元素，保留原幻灯片内容 | P0 |
| 批量处理 | 支持一次性上传多个 PPTX 文件批量处理 | P0 |
| 文件下载 | 处理完成后支持打包下载（ZIP） | P0 |

### 2.2 扩展功能（E）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 预置水印库 | 内置常见 AI PPT 工具水印特征 | P1 |
| 自定义规则 | 用户可添加自定义水印关键词或图片进行匹配 | P1 |
| 水印类型识别 | 识别并分类：水印类型（文字/图片）、位置、来源 | P2 |
| 预览确认 | 处理前展示检测到的水印，用户确认后清除 | P2 |
| 清理报告 | 生成清理报告（移除了多少水印、处理了多少页） | P2 |

---

## 3. 水印来源支持

### 3.1 预置水印库（P1）

| 来源 | 特征 |
|------|------|
| NotebookLM | "NotebookLM" 文字水印、Logo 图片 |
| Gamma.app | "Made with Gamma"、Gamma Logo |
| Beautiful.ai | "beautiful.ai" 文字、Logo |
| Canva PPT | Canva Logo |
| AiPPT | AiPPT 相关文字 |
| 通义万相 | 阿里系相关文字/Logo |
| 讯飞智文 | 讯飞相关文字/Logo |

### 3.2 自定义规则（P1）

| 规则类型 | 说明 |
|----------|------|
| 关键词 | 用户输入文字，水印包含该文字即清除 |
| 图片匹配 | 用户上传水印图片，匹配相同/相似图片 |
| 颜色特征 | 按透明度/颜色识别（高级） |

---

## 4. 技术方案

### 4.1 文件格式

**输入**：`.pptx` 文件（Microsoft PowerPoint 2007+）  
**输出**：清理后的 `.pptx` 文件

### 4.2 水印识别策略

1. **文字水印**
   - 检测半透明文字（透明度 < 50%）
   - 检测斜向排列文字（旋转角度 45° 或 -45°）
   - 关键词匹配：预置库 + 自定义关键词

2. **图片水印**
   - 检测小尺寸重复图片（Logo 类）
   - 检测全屏半透明叠加图片
   - 图像相似度匹配（自定义图片）

3. **母版水印**
   - 遍历 `ppt/slides/_rels/*.xml.rels` 关联的母版
   - 检查母版中的水印元素

### 4.3 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端 | Python + python-pptx + Pillow + fastapi |
| 前端 | 简单 HTML/JS 或 Streamlit 快速原型 |
| 部署 | 可本地运行或部署为 Web 服务 |

## 6. 验收标准

1. 上传 1 个带 NotebookLM 水印的 PPTX，输出干净的文件
2. 批量上传 10 个 PPTX 文件，全部成功处理
3. 处理后的文件用 PowerPoint / WPS 打开正常显示
4. 原文件内容（文字、图片、布局）完整保留
5. 自定义关键词规则可正确识别并清除
